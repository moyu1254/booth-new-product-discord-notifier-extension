# BOOTH New Product Discord Notifier Extension v0.2.1

Security and maintenance release.

## Changes

- Store Discord Webhook URL settings in local extension storage instead of synced storage.
- Migrate existing synced settings to local storage and remove the synced copy.
- Validate Discord Webhook URLs before saving or sending notifications.
- Share BOOTH product parsing logic between background and offscreen contexts.
- Exclude unnecessary `.gitignore` from Chromium release packages.
- Limit GitHub Actions write permissions to the tag release job only.
